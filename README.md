## ✨ Key Features & Learning Areas
* **🏗️ Page Object Model (POM):** Scalable, clean, and maintainable architecture.
* **📡 API Testing Integration:** Combining UI and API testing seamlessly in a single test runner.
* **📊 Advanced Reporting:** HTML reports, Allure reporting, and full tracing setup.
* **🤖 Interactive Debugging:** Leveraging Playwright UI mode, Trace Viewer, and VS Code extension.
* **🔄 CI/CD Readiness:** Pre-configured GitHub Actions workflow for automated test execution on push/PR.
* **📸 Visual Regression:** Snapshot and screenshot testing for visual validation.
